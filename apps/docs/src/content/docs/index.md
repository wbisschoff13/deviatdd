This site holds the documentation for the DeviaTDD framework. It is organised by the [Diátaxis](https://diataxis.fr/) framework — four quadrants that answer four different reader questions — and every page declares which quadrant it belongs to via its `doc_type` frontmatter. Each quadrant has its own landing page at `<quadrant>/index.md`; that landing page is the navigation pivot for the quadrant and is the first entry in the sidebar for that quadrant. Pick the register that matches your immediate need below.

## Quadrants

## Tutorials

Learning-oriented. Walk through a complete end-to-end exercise.
→ [Tutorials index](./tutorials/index.md)

## How-To

Task-oriented. Recipe for one well-defined job.
→ [How-To index](./how-to/index.md)

## Reference

Information-oriented. Tables of fields, flags, and configuration options.
→ [Reference index](./reference/index.md)

## Explanation

Understanding-oriented. Why-frames and design trade-offs.
→ [Explanation index](./explanation/index.md)
